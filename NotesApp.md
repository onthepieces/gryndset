# 📝 Notes Organizer

The **Notes Organizer** provides a distraction-free editing canvas featuring structured folders, search options, and live split-screen Markdown rendering.

---

## 📁 Organizing Notes

* **Folders**: Group notes into separate folders (e.g. `General`, `Client Meetings`, `Specifications`).
* **Search Filters**: Type key terms inside the folder sidebar to filter notes in real-time.
* **Pinning**: Toggle the pin icon on critical notes. Pinned notes are anchored at the top of your list for instant access.

---

## ✍️ Split-Screen Markdown Editor

* **Live Rendering**: Write standard markdown syntax in the left panel. The right panel parses and displays the HTML output in real time.
* **Layout Responsiveness**:
  * **Desktop viewports**: Side-by-side editing and rendering split.
  * **Mobile viewports**: The app adopts a focused toggle flow:
    * Tap a note to open its editor full-width.
    * Use the **Back** button to return to the folder listing safely.
